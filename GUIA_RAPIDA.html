📝 Cómo Crear Registros

Crear un Nuevo Torneo

  • Haz clic en el botón "+ Nuevo Torneo" en la página de Torneos
  • O usa el botón FAB (+) en la esquina y selecciona "Crear Nuevo Torneo"
  • Completa los datos del formulario:
    • ✓ Nombre del torneo
    • ✓ Categoría (Primera, Sub-20, etc)
    • ✓ Formato (Liga, Eliminación, Grupos)
    • ✓ Fechas de inicio y fin
    • ✓ Máximo de equipos
  • Haz clic en "Guardar"

Registrar un Equipo

  • Ve a la sección "Equipos"
  • Haz clic en "+ Nuevo Equipo"
  • Completa:
    • ✓ Nombre del equipo
    • ✓ Color identificador
    • ✓ Categoría
    • ✓ Delegado y teléfono
  • Guarda

Agregar Jugadores

  • Ve a la sección "Jugadores"
  • Haz clic en "+ Nuevo Jugador"
  • Rellena:
    • ✓ Nombre y apellido
    • ✓ Equipo al que pertenece
    • ✓ Posición (Arquero, Defensor, etc)
    • ✓ Edad
  • Confirma

💾 Persistencia de Datos

La aplicación utiliza localStorage para guardar todos los datos:

✅ Los datos se guardan automáticamente cuando creas o modificas registros
⚠️ Los datos se guardan localmente en tu navegador. Si limpias el historial, se pueden perder.

Estructura de datos:

Elemento Almacenado Como Descripción
Torneos appData.torneos[] Lista de campeonatos
Equipos appData.equipos[] Registro de equipos
Jugadores appData.jugadores[] Base de jugadores
Partidos appData.partidos[] Fixture y resultados
Posiciones appData.posiciones[] Tabla de clasificación
